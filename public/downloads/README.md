# Downloads Directory

Place your installer files here:

## Windows
- `windows/MindMesh-Setup.exe` - Windows installer

## macOS
- `mac/MindMesh.dmg` - macOS disk image

## Notes

- File sizes are displayed in the download component
- Update file sizes in `components/DownloadSection.tsx` if they change
- Ensure files are properly signed for distribution
- Consider adding checksums for verification
